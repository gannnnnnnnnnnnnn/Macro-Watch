# Macro-Watch Project Brief

Macro-Watch is a personal macro research cockpit.

The first version should focus on:
- a polished market dashboard
- OpenStock UI research
- OpenBB data and AI capability research
- a future Trader Reader for importing and reading trader PDFs
- a long-term path toward macro state, financial stress, and AI-assisted research

Out of scope for v0:
- auto trading
- broker integration
- automatic thesis validation
- multi-trader scoring
